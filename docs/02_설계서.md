## 1. 시스템 아키텍처

### 1.1 전체 구조

┌─────────────────────────────────────────────────┐
│                  Flutter App                     │

## 2. 상태 관리 설계

### 2.1 Riverpod 프로바이더 구조

#### 가계부 프로바이더 (5개)

```
hiveBoxProvider (Provider<Box>)
    │
    └── transactionsProvider (StateNotifierProvider<TransactionsNotifier, List<Transaction>>)
            │
            ├── transactionsForDayProvider (Provider.family<List<Transaction>, DateTime>)
            │
            ├── monthlyExpenseProvider (Provider.family<int, DateTime>)
            │
            └── monthlyIncomeProvider (Provider.family<int, DateTime>)
```

| Provider | 타입 | 역할 |
|----------|------|------|
| `hiveBoxProvider` | `Provider<Box>` | Hive 'transactions' 박스 주입 |
| `transactionsProvider` | `StateNotifierProvider` | 전체 거래 목록 CRUD |
| `transactionsForDayProvider` | `Provider.family` | 특정 날짜 거래 필터링 |
| `monthlyExpenseProvider` | `Provider.family` | 월간 지출 합계 |
| `monthlyIncomeProvider` | `Provider.family` | 월간 수입 합계 |

#### 영어 학습 프로바이더 (8개)

```
learningBoxProvider (Provider<Box>)
    │
    └── learningProgressProvider (StateNotifierProvider<..., Map<String, LearningProgress>>)
            │
            └── learnedCountProvider (Provider<int>)

wordBankProvider (Provider<List<EnglishWord>>)
    │
    └── wordsByCategoryProvider (Provider.family<List<EnglishWord>, WordCategory?>)

totalWordsProvider (Provider<int>)

transactionsProvider ──┬── recentEnglishExpressionsProvider (Provider<List<TranslatedTransaction>>)
                       │
                       └── quizQuestionsProvider (Provider<List<QuizQuestion>>)
                            ↑
                       wordBankProvider
```

| Provider | 타입 | 역할 |
|----------|------|------|
| `learningBoxProvider` | `Provider<Box>` | Hive 'learning_progress' 박스 주입 |
| `wordBankProvider` | `Provider` | 38개 단어 목록 제공 |
| `wordsByCategoryProvider` | `Provider.family` | 카테고리별 필터링 |
| `learningProgressProvider` | `StateNotifierProvider` | 학습 진도 관리 |
| `learnedCountProvider` | `Provider` | 학습 완료 단어 수 |
| `totalWordsProvider` | `Provider` | 전체 단어 수 |
| `recentEnglishExpressionsProvider` | `Provider` | 최근 거래 영어 번역 |
| `quizQuestionsProvider` | `Provider` | 퀴즈 문제 생성 |

### 2.2 StateNotifier 설계

#### TransactionsNotifier
```dart
class TransactionsNotifier extends StateNotifier<List<Transaction>> {
  final Box _box;

  TransactionsNotifier(this._box) : super([]) {
    _loadFromHive();   // 박스에서 기존 데이터 로드
    if (state.isEmpty) _initSampleData();  // 초기 샘플 데이터
  }

  void add(Transaction tx)     // 추가 + Hive 저장 + 날짜순 정렬
  void update(Transaction tx)  // ID 매칭 후 교체 + Hive 업데이트
  void delete(String id)       // ID로 삭제 + Hive 삭제
}
```

#### LearningProgressNotifier
```dart
class LearningProgressNotifier extends StateNotifier<Map<String, LearningProgress>> {
  final Box _box;

  LearningProgressNotifier(this._box) : super({}) {
    _loadFromHive();
  }

  void markCorrect(String wordId)      // 정답 카운트 증가
  void markIncorrect(String wordId)    // 시도 카운트만 증가
  void toggleBookmark(String wordId)   // 북마크 토글
}
```

## 3. 라우팅 설계

### 3.1 GoRouter 라우트 테이블

| 경로 | 페이지 | 파라미터 | 설명 |
|------|--------|----------|------|
| `/` | LedgerMainPage | - | 가계부 메인 (캘린더) |
| `/add` | ManualEntryPage | extra: DateTime? | 신규 거래 등록 |
| `/edit` | ManualEntryPage | extra: Transaction | 거래 수정 |
| `/day/:date` | DayDetailPage | pathParam: date (yyyyMMdd) | 일별 상세 |
| `/ocr` | OcrScanPage | - | 영수증 스캔 |
| `/english` | EnglishLearningPage | - | 영어 학습 대시보드 |
| `/english/words` | WordListPage | - | 단어장 |
| `/english/quiz` | QuizPage | - | 퀴즈 |

### 3.2 네비게이션 흐름
```
LedgerMainPage (/)
├── [날짜 선택] → DayDetailPage (/day/:date)
│   ├── [FAB] → ManualEntryPage (/add)
│   ├── [카드 탭] → ManualEntryPage (/edit)
│   └── [롱프레스] → 삭제 다이얼로그
├── [FAB] → ManualEntryPage (/add)
├── [카메라 아이콘] → OcrScanPage (/ocr)
└── [책 아이콘] → EnglishLearningPage (/english)
    ├── [단어장 카드] → WordListPage (/english/words)
    └── [퀴즈 카드] → QuizPage (/english/quiz)
        └── [다시 풀기] → QuizPage (재생성)
```

## 4. 데이터 영속화 설계

### 4.1 Hive 박스 구조

#### transactions 박스
```
Key: String (Transaction.id)
Value: Map<String, dynamic>
  {
    'id': String,
    'title': String,
    'date': String (ISO 8601),
    'amount': int,
    'type': String ('expense' | 'income'),
    'iconCodePoint': int,
    'iconFontFamily': String,
    'iconColorValue': int,
    'category': String?
  }
```

#### learning_progress 박스
```
Key: String (wordId)
Value: Map<String, dynamic>
  {
    'wordId': String,
    'correctCount': int,
    'totalAttempts': int,
    'lastStudied': String (ISO 8601),
    'isBookmarked': bool
  }
```

### 4.2 초기화 시퀀스
```
main()
  → WidgetsFlutterBinding.ensureInitialized()
  → Hive.initFlutter()
  → await Hive.openBox('transactions')
  → await Hive.openBox('learning_progress')
  → runApp(ProviderScope(overrides: [hiveBoxProvider, learningBoxProvider]))
```

## 5. UI 설계

### 5.1 테마 시스템

| 토큰 | 값 | 용도 |
|------|-----|------|
| `AppColors.accent` | `#5C6BC0` (Indigo 400) | 주요 액센트 |
| `AppColors.accentDark` | `#3949AB` (Indigo 700) | 그래디언트 끝 |
| `AppColors.income` | `#66BB6A` (Green 400) | 수입 색상 |
| `AppColors.expense` | `#EF5350` (Red 400) | 지출 색상 |
| `AppColors.surface` | `#FFFFFF` | 카드 배경 |
| `AppColors.textPrimary` | `#212121` | 기본 텍스트 |
| `AppColors.textSecondary` | `#757575` | 보조 텍스트 |

### 5.2 공유 데코레이션
- `AppTheme.cardShadow`: 카드 그림자 (blur 12, offset 0,4)
- `AppTheme.headerGradient`: 헤더 그래디언트 (accent → accentDark)
- `AppTheme.light`: MaterialApp ThemeData (Material 3, 한글 폰트)

### 5.3 위젯 계층 패턴
```
Page (ConsumerWidget / ConsumerStatefulWidget)
  └── Scaffold
      ├── AppBar (제목, 뒤로가기)
      ├── Body
      │   └── SingleChildScrollView / ListView
      │       └── _SectionWidget (private 위젯)
      └── FloatingActionButton? (선택)
```
