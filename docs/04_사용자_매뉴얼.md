# Pay N Say - 사용자 매뉴얼

**앱 이름:** Pay N Say
**버전:** 1.0
**작성일:** 2026-03-18

---

## 1. 앱 소개

Pay N Say는 가계부 관리와 영어 학습을 결합한 앱입니다. 일상 소비를 기록하면서 관련 영어 표현을 자연스럽게 배울 수 있습니다.

---

## 2. 가계부 사용법

### 2.1 메인 화면 (월간 캘린더)

앱을 실행하면 월간 캘린더가 표시됩니다.

**화면 구성:**
- **상단 헤더:** 현재 월의 수입, 지출, 잔액 표시
- **캘린더:** 날짜별 지출 여부를 점(dot)으로 표시
- **하단 FAB(+):** 새 내역 추가 버튼

**조작 방법:**
- 좌우 스와이프: 이전/다음 달 이동
- 날짜 탭: 해당 날짜의 상세 내역 보기
- (+) 버튼: 새 거래 입력 화면으로 이동
- 📖 아이콘 (헤더 우측): 영어 학습 화면으로 이동

### 2.2 일별 상세 화면

선택한 날짜의 거래 내역을 확인하고 관리합니다.

**화면 구성:**
- **앱바:** "M월 D일 (요일)" 형식
- **건수 배지:** 해당 날의 내역 수
- **요약 카드:** 수입 / 지출 / 잔액
- **거래 목록:** 상호명, 금액, 카테고리 아이콘

**조작 방법:**
- 거래 카드 탭: 수정 화면으로 이동
- 거래 카드 길게 누르기: 삭제 다이얼로그 표시
- (+) 버튼: 해당 날짜로 새 내역 추가

**내역이 없는 날:**
- "이 날의 내역이 없어요" 안내 메시지 표시
- "+ 버튼으로 내역을 추가해보세요" 안내

### 2.3 새 거래 입력

**입력 순서:**
1. **지출/수입 선택:** 상단 토글에서 선택 (기본: 지출)
2. **상호명 입력:** 예) 스타벅스, 쿠팡, 3월 급여
3. **카테고리 선택:** 드롭다운에서 선택 (식비, 카페, 교통 등)
4. **날짜 선택:** 달력 아이콘을 눌러 날짜 변경 (기본: 오늘)
5. **금액 입력:** 숫자만 입력 (₩ 접두사 자동 표시)
6. **저장:** "지출 저장하기" 또는 "수입 저장하기" 버튼

**유효성 검사:**
- 상호명 미입력 시: "상호명을 입력해 주세요." 오류
- 금액 미입력 시: "금액을 입력해 주세요." 오류

### 2.4 거래 수정

일별 상세 화면에서 거래 카드를 탭하면 수정 화면으로 이동합니다.
- 기존 값이 모든 필드에 미리 채워짐
- 앱바 제목: "내역 수정"
- 저장 버튼: "수정 완료"

### 2.5 거래 삭제

일별 상세 화면에서 거래 카드를 **길게 누르면** 삭제 다이얼로그가 표시됩니다.
- "삭제" 버튼: 해당 내역 즉시 삭제
- "취소" 버튼: 다이얼로그 닫기, 변경 없음

## 3. 영어 학습 사용법

### 3.1 영어 학습 진입

가계부 메인 화면 상단 헤더의 📖(책) 아이콘을 탭합니다.

### 3.2 영어 학습 대시보드

**화면 구성:**

1. **학습 진도 카드**
   - "학습한 단어: N / M" 표시
   - 프로그레스 바로 진행률 시각화
   - 단어를 퀴즈에서 3번 이상 맞추면 "학습 완료"로 카운트

2. **오늘의 영어 표현**
   - 최근 7일간의 거래 내역을 영어 문장으로 변환
   - 예: "I spent ₩6,500 on coffee." (커피에 ₩6,500을 썼어요.)
   - 최대 5건 표시

3. **학습 메뉴**
   - 단어장 카드: 탭하여 단어장으로 이동
   - 퀴즈 카드: 탭하여 퀴즈로 이동

### 3.3 단어장

38개의 소비/금융 관련 영어 단어를 학습합니다.

**카테고리 (9개):**
| 카테고리 | 단어 수 | 예시 |
|----------|---------|------|
| 음식 | 5 | lunch, dinner, grocery shopping |
| 카페 | 4 | coffee, latte, dessert |
| 쇼핑 | 4 | online shopping, clothes |
| 뷰티 | 2 | haircut, cosmetics |
| 문화 | 3 | movie ticket, book |
| 교통 | 4 | subway fare, taxi |
| 의료 | 2 | hospital, pharmacy |
| 금융 | 10 | budget, savings, deposit |
| 수입 | 4 | salary, allowance |

**사용 방법:**
1. 상단 카테고리 칩을 좌우 스크롤하여 필터링
2. "전체"를 선택하면 모든 단어 표시
3. 단어 카드를 탭하면 확장하여 예문 표시
4. ♥ 아이콘: 북마크 토글
5. "학습 완료" 배지: 퀴즈에서 3번 이상 정답 시 표시

### 3.4 퀴즈

**퀴즈 진행:**
1. 총 10문제, 4지선다
2. 문제 유형:
   - 한국어 → English (한국어 뜻을 보고 영어 선택)
   - English → 한국어 (영어를 보고 한국어 뜻 선택)
3. 보기 중 하나를 탭하여 답 선택
4. 정답: 초록색으로 표시
5. 오답: 빨간색으로 표시 + 정답 초록색 표시
6. 자동으로 다음 문제로 이동

**결과 화면:**
- 최종 점수: "N / 10"
- 백분율 표시
- 오답 리뷰: 틀린 문제의 정답 확인
- "다시 풀기" 버튼: 새로운 문제로 재시작

**학습 진도:**
- 같은 단어를 3번 이상 맞추면 "학습 완료" 처리
- 진도는 앱을 종료해도 유지됨 (로컬 저장)

## 4. 데이터 관리

### 4.1 데이터 저장
- 모든 데이터는 기기 내부에 저장됩니다 (로컬 전용).
- 네트워크 연결 없이 모든 기능을 사용할 수 있습니다.
- 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

### 4.2 초기 샘플 데이터
- 앱 최초 실행 시 9건의 샘플 거래 내역이 포함되어 있습니다.
- 샘플 데이터는 자유롭게 수정/삭제할 수 있습니다.

## 5. 화면 이동 요약

| 현재 화면 | 동작 | 이동할 화면 |
|-----------|------|-------------|
| 가계부 메인 | 날짜 탭 | 일별 상세 |
| 가계부 메인 | (+) 버튼 | 새 거래 입력 |
| 가계부 메인 | 📖 아이콘 | 영어 학습 대시보드 |
| 일별 상세 | 카드 탭 | 거래 수정 |
| 일별 상세 | 카드 길게 누르기 | 삭제 다이얼로그 |
| 일별 상세 | (+) 버튼 | 새 거래 입력 |
| 영어 학습 | 단어장 카드 | 단어장 |
| 영어 학습 | 퀴즈 카드 | 퀴즈 |
| 퀴즈 결과 | 다시 풀기 | 퀴즈 (새 문제) |
| 모든 하위 화면 | ← 뒤로가기 | 이전 화면 |

## 6. 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| 저장이 안 됨 | 상호명과 금액을 모두 입력했는지 확인 |
| 영어 표현이 비어 있음 | 최근 7일 내 거래가 없으면 표시되지 않음 |
| 학습 완료가 안 됨 | 같은 단어를 퀴즈에서 3번 이상 맞춰야 함 |
| 데이터 초기화 원할 때 | 앱 삭제 후 재설치 |
