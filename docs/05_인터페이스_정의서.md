## 1. 내부 인터페이스 (프로바이더 API)

### 1.1 가계부 프로바이더

#### hiveBoxProvider
```dart
final hiveBoxProvider = Provider<Box>((ref) => throw UnimplementedError());
```
- **설명:** Hive 'transactions' 박스 주입용 프로바이더
- **오버라이드:** main.dart에서 실제 Box 인스턴스로 오버라이드
- **사용처:** TransactionsNotifier 내부

---

#### transactionsProvider
```dart
final transactionsProvider =
    StateNotifierProvider<TransactionsNotifier, List<Transaction>>((ref) {
  final box = ref.watch(hiveBoxProvider);
  return TransactionsNotifier(box);
});
```

**Notifier 메서드:**

| 메서드 | 시그니처 | 설명 |
|--------|----------|------|
| `add` | `void add(Transaction tx)` | 거래 추가. Hive 저장 + 날짜 내림차순 정렬 |
| `update` | `void update(Transaction tx)` | ID 매칭하여 거래 수정. Hive 동기화 |
| `delete` | `void delete(String id)` | ID로 거래 삭제. Hive 동기화 |

**상태 (List\<Transaction\>):**
- 날짜 내림차순 정렬 유지
- 초기 로드: Hive에서 복원 또는 샘플 9건 생성

---

## 2. 데이터 모델 인터페이스

### 2.1 Transaction

```dart
class Transaction {
  final String id;
  final String title;
  final DateTime date;
  final int amount;
  final TransactionType type;
  final IconData icon;
  final Color iconColor;
  final String? category;

  Transaction copyWith({...});
  Map<String, dynamic> toMap();
  factory Transaction.fromMap(Map<String, dynamic> map);
}

enum TransactionType { expense, income }
```

```json
  "iconCodePoint": 59391,
  "iconFontFamily": "MaterialIcons",
  "iconColorValue": 4283315246,
```

```dart
}
```

### 2.3 LearningProgress

```dart
class LearningProgress {
  final String wordId;
  final int correctCount;
  final int totalAttempts;
  final DateTime lastStudied;
  final bool isBookmarked;

  bool get isLearned => correctCount >= 3;
  double get accuracy => totalAttempts > 0 ? correctCount / totalAttempts : 0;

  LearningProgress copyWith({...});
  Map<String, dynamic> toMap();
  factory LearningProgress.fromMap(Map<String, dynamic> map);
}
```

**직렬화 형식 (toMap):**
```json
{
  "wordId": "food_lunch",
  "correctCount": 3,
  "totalAttempts": 5,
  "lastStudied": "2026-03-18T14:30:00.000",
  "isBookmarked": true
}
```

## 3. 유틸리티 인터페이스

### 3.1 TransactionTranslator

```dart
class TransactionTranslator {
  /// 거래를 영어 문장으로 변환
  /// 지출: "I spent ₩{amount} on {item}."
  /// 수입: "I earned ₩{amount} from {source}."
  static String translate(Transaction tx);

  /// 거래를 한국어 문장으로 변환
  /// 지출: "{item}에 ₩{amount}을 썼어요."
  /// 수입: "{source}에서 ₩{amount}을 벌었어요."
  static String translateKorean(Transaction tx);
}
```

## 4. 라우팅 인터페이스

### 4.1 GoRouter 네비게이션 API

| 호출 | 설명 |
|------|------|
| `context.push('/day/${format(date)}')` | 일별 상세 이동 (yyyyMMdd) |
| `context.push('/add', extra: DateTime)` | 입력 화면 (날짜 전달) |
| `context.push('/edit', extra: Transaction)` | 수정 화면 (거래 전달) |
| `context.push('/ocr')` | OCR 스캔 이동 |
| `context.push('/english')` | 영어 학습 이동 |
| `context.push('/english/words')` | 단어장 이동 |
| `context.push('/english/quiz')` | 퀴즈 이동 |
| `context.pop()` | 이전 화면 복귀 |
| `context.pop(true)` | 이전 화면 복귀 + 결과 전달 |

## 5. Hive 저장소 인터페이스

### 5.1 Box API 사용 패턴

```dart
// 저장
await box.put(transaction.id, transaction.toMap());

// 조회
final map = box.get(id) as Map?;
if (map != null) Transaction.fromMap(Map<String, dynamic>.from(map));

// 삭제
await box.delete(id);

// 전체 로드
for (final key in box.keys) {
  final map = box.get(key) as Map;
  transactions.add(Transaction.fromMap(Map<String, dynamic>.from(map)));
}
```
