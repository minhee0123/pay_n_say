## 1. 앱 초기화 흐름

```
┌──────────────┐
│  앱 실행      │
└──────┬───────┘
       ▼
┌──────────────────────────────┐
│  WidgetsFlutterBinding       │
│  .ensureInitialized()        │
└──────┬───────────────────────┘
       ▼
┌──────────────────────────────┐
│  Hive.initFlutter()          │
└──────┬───────────────────────┘
       ▼
┌──────────────────────────────┐
│  await Hive.openBox           │
│  ('transactions')             │
└──────┬───────────────────────┘
       ▼
┌──────────────────────────────┐
│  await Hive.openBox           │
│  ('learning_progress')        │
└──────┬───────────────────────┘
       ▼
┌──────────────────────────────┐
│  runApp(ProviderScope(        │
│    overrides: [               │
│      hiveBoxProvider,         │
│      learningBoxProvider      │
│    ]                          │
│  ))                           │
└──────┬───────────────────────┘
       ▼
```

## 2. 거래 입력 흐름

```
┌──────────────┐
│  사용자: (+)  │
│  버튼 탭      │
└──────┬───────┘
       ▼
┌──────────────────┐
│  ManualEntryPage  │
│  (/add) 이동      │
└──────┬───────────┘
       ▼
┌──────────────────┐
│  지출/수입 선택   │
│  (기본: 지출)     │
└──────┬───────────┘
       ▼
┌──────────────────┐
│  폼 입력          │
│  - 상호명         │
│  - 카테고리       │
│  - 날짜           │
│  - 금액           │
└──────┬───────────┘
       ▼
┌──────────────────┐
│  저장 버튼 탭     │
└──────┬───────────┘
       ▼
┌───────────────────────┐
│  유효성 검사           │
│  - 상호명 비어있나?    │  ──Yes──→ "상호명을 입력해 주세요." 표시
│  - 금액 비어있나?      │  ──Yes──→ "금액을 입력해 주세요." 표시
└──────┬────────────────┘
       │ 통과
       ▼
┌──────────────────────────┐
│  Transaction 객체 생성    │
│  (UUID, 입력값, 아이콘)   │
└──────┬───────────────────┘
       ▼
┌──────────────────────────┐
│  notifier.add(tx)         │
│  → state에 추가           │
│  → 날짜 내림차순 정렬     │
│  → box.put(id, toMap())   │
└──────┬───────────────────┘
       ▼
┌──────────────────────────┐
│  context.pop(true)        │
│  이전 화면 복귀           │
│  → 파생 프로바이더 자동   │
│    갱신 (월간합계, 일별)  │
└──────────────────────────┘
```

## 3. 거래 수정 흐름

```
┌──────────────────┐
│  일별 상세 화면   │
│  거래 카드 탭     │
└──────┬───────────┘
       ▼
┌──────────────────────────┐
│  ManualEntryPage (/edit)  │
│  extra: Transaction       │
│  → 기존 값 폼에 채움      │
│  → 앱바: "내역 수정"      │
│  → 버튼: "수정 완료"      │
└──────┬───────────────────┘
       ▼
┌──────────────────┐
│  사용자 수정 후   │
│  "수정 완료" 탭   │
└──────┬───────────┘
       ▼
┌──────────────────────────┐
│  유효성 검사 통과         │
└──────┬───────────────────┘
       ▼
┌──────────────────────────┐
│  notifier.update(tx)      │
│  → ID 매칭하여 교체       │
│  → box.put(id, toMap())   │
└──────┬───────────────────┘
       ▼
┌──────────────────────────┐
│  context.pop(true)        │
│  이전 화면으로 복귀       │
└──────────────────────────┘
```

## 8. 데이터 흐름 (Riverpod 반응형)

```
사용자 액션
    │
    ▼
┌────────────────────┐
│ Notifier 메서드     │  (add, update, delete, markCorrect ...)
│ 호출               │
└────────┬───────────┘
         │
    ┌────┴────┐
    ▼         ▼
┌────────┐ ┌──────────────┐
│ Hive   │ │ state 갱신    │
│ 저장   │ │ (불변 교체)   │
└────────┘ └──────┬───────┘
                  │
                  ▼
         ┌────────────────────┐
         │ 파생 프로바이더     │
         │ 자동 재계산         │
         │                    │
         │ • transactionsForDay│
         │ • monthlyExpense    │
         │ • monthlyIncome     │
         │ • learnedCount      │
         │ • recentExpressions │
         │ • quizQuestions     │
         └────────┬───────────┘
                  │
                  ▼
         ┌────────────────────┐
         │ ref.watch()로      │
         │ 구독 중인 위젯     │
         │ 자동 리빌드         │
         └────────────────────┘
```

## 9. 전체 화면 전환 흐름도

```
                         ┌─────────────────┐
                         │   앱 시작        │
                         └────────┬────────┘
                                  │
                                  ▼
                    ┌─────────────────────────┐
                    │  ① 가계부 메인 (/)       │
                    │  월간 캘린더 + 요약 헤더  │
                    └──┬───┬───┬───┬──────────┘
                       │   │   │   │
          ┌────────────┘   │   │   └──────────────┐
          │                │   │                   │
          ▼                ▼   ▼                   ▼
  ┌───────────────┐ ┌──────────┐ ┌──────────┐ ┌──────────────────┐
  │② 일별 상세    │ │③ 입력    │ │⑤ OCR    │ │⑥ 영어학습       │
  │(/day/:date)   │ │(/add)    │ │(/ocr)   │ │(/english)        │
  └──┬─────┬──────┘ └──────────┘ └──────────┘ └──┬──────┬───────┘
     │     │                                      │      │
     ▼     ▼                                      ▼      ▼
  ┌──────┐┌──────┐                         ┌──────────┐┌─────────┐
  │③입력 ││④수정 │                         │⑦ 단어장  ││⑧ 퀴즈  │
  │(/add)││(/edit)│                         │(/english ││(/english│
  │      ││      │                         │ /words)  ││ /quiz)  │
  └──────┘└──────┘                         └──────────┘└─────────┘

  ← : 모든 하위 화면은 뒤로가기(pop)로 상위 화면 복귀
```
